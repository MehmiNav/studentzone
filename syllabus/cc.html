Compiler Construction

Compiler Construction

It is a process of constructing a Compiler.It is also Known as translator.It means, it converts the high level language into low level language.

Compiler

A compiler is a computer program which helps you transform source code written in HLL into LL machine language .It translate the code written in one programming language to some other language without chaning the meaning of code the compiling process include basic translation mechanism and error detection.The compiler process goes through Lexical(tokens), Syntax & Semantic analysis at the front end and code generation and optimization at the backend.

Types of Compiler

1.Single Pass Compiler

Single pass compiler:- In this compiler source code directly transforms into machine code.It is also known as narrow compiler.
For Example:-Pascal Language.

2.Multipass Compiler

Multipass Compiler:- The multipass compiler process the source code or syntax tree of a program several times.It divide a large program into multiple small programs and process them.It develops multiple intermediate codes.All of these multipass take the output of the previous phase as an input.So,it requires less memory.It is also known as 'Wide Compiler'.

Related item

HLL

It stands for High Level Language . HLL is a programming language such as C , FORTRAN or PASCAL that enable a programmer to write a prgrams that are more or less independent of a particular type of computer . Such language are consider high level because they are closer to human language and further from machine language.


LLL

LLL stands for Low Level Language . LLL are the language that sit close to the computers instructions set . An instruction set is a set of instructions that the processor understand .
Two types of LLL are Machine Code and Assembly Code.

Pass

It is an implementation of compiler one or more phases are combined into a module called a pass .
